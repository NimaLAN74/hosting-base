# D14 — Clustering Dataset Specification  
Version 1.0

## 1. Use Case
Group regions into similar energy/geo profiles.

## 2. Feature Set (examples)
- Energy mix by product (share of total final consumption)
- Share of renewables in electricity generation
- Final energy consumption per capita (if population available externally)
- Industrial_density (km² or count per km²)
- Building_density (buildings per km²)
- Land-use ratios (urban vs non-urban share)

## 3. Output
- Cluster label per region
- Optional cluster-level summaries (centroids, profiles).
